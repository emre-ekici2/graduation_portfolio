Estate Management System - Project 4.0

This project was created for the Project PHP course during my studies. We were asked to develop an estate management system for a family with multiple estates and workers. The goal was to create a centralized application so that the family and the workers could manage their schedules, meals, work shifts and meeting with their special needs grandchild easier.

Technologies used:

  • PHP
  • Laravel
  • Tailwind

My contributions:

  • Set up the base architecture for the project including the routing and authorization for certain pages
  • Created the livewire components to be used for the application
  • Developed the functionalities to manage user accounts
  • Created the meal management system
  • Created the data models
  • Developed the FAQ pages

My thoughts:

This project was the first project where we had to put aside many hours at a time to work on it. Compared to our other projects, we had no true guide or basis to follow for this project thus making it the first one where we had to truly improvise and come up with our own ideas and implementations. This project was a success in my opinion as it was a learning experience for working with a larger group of people (5 people in total) in an agile manner. I was able to participate in client meetings, plan out sprints and develop other important skills which will be needed in my future professional career.
